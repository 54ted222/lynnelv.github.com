js函数作用域

与其它语言（java,php）不同，js不使用块级作用域，而是使用函数作用域,即在js的函数内声明的变量在函数体的任何地方可见。

js函数定义的常见方式

1.通过function语句

	function test() {
		//在js解析的时候，所有var声明的变量和function函数,包括函数名和函数体，
		//所以可以在声明前使用，都会被提前到脚本或函数顶部，所以它们在整个脚
		//本或函数中的任何位置都可见，但是变量的定义，则在定义它们的位置执行，
		//因此在定义前引用变量会出现undefined
	}

2.通过变量赋值

	var test = function() {
		//将变量test指向匿名函数，相当于
		//var test;
		//test();	//此处调用会报错，因为test只声明未定义
		//test = function() {}
	}

好的编码习惯应该是，把脚本或函数中的所有变量声明在脚本或函数开头；函数先声明后调用。所以第二种方式为推荐方式。

P.S：函数声明会覆盖变量声明，但不会覆盖变量赋值

	function test(){
		return 1;
	}
	var test;
	alert(typeof test);    //"function"

函数声明的优先级高于变量声明的优先级，但是如果变量赋值了，则不会被覆盖。

	function test(){
	    return 1;
	}
	var test = 1;
	alert(typeof test);    //"number"
